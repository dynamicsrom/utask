========================================================================
    DYNAMIC LINK LIBRARY : UTaskService Project Overview
========================================================================
	
	UTaskService is the most important part of uTask project.
	While UTaskHook is just a supporting library, service part 
	does everything with menu_workers.
	
	Main functions:
	- getting signals from hook part.
	- hooking menu_workers.
	- exposing an internal interface to control different options from
	external applications.
