========================================================================
    DYNAMIC LINK LIBRARY : UTaskHook Project Overview
========================================================================
	
	UTaskHook is a supporting part of uTask project.
	
	Functions:
	-hooking CreateWindowExW and other functions
	-sending messages with new HWNDs to the service part of project
	-hooking windows' moving and fixing their position
	
	Attention! This part can't be unloaded at the moment, so any changes
	will require soft reset.
